  • 1、背景
  • 2、示例

1、背景

Trackbar：轨迹条、跟踪条、滑动条。在本博客中取滑动条的翻译。

opencv提供了createTrackbar函数来创建滚动条，该进度条可以在UI界面上进行滑动，
当滑动的时候，会有不同的值。
通过该功能 ，可以让用户在UI界面实现一些交互功能，比如图像的亮度、饱和度等参数调整，视频的进度调整等。

cv::createTrackbar函数原型如下：

int createTrackbar(const string& trackbarname,  					const string&winname,  					int* value,   					int count ， 					TrackbarCallback onChange = 0,  					void* userdata = 0)<
